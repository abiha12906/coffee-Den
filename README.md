 ☕ Coffee Den

Coffee Den is a frontend coffee shop web application built using HTML, CSS, and JavaScript. It provides a modern and interactive user interface that simulates an online coffee ordering experience.

 🚀 Features
- User login/signup using localStorage
- Add to cart with quantity control
- Dynamic total calculation with delivery charges
- Product detail modals
- Image carousel
- Checkout simulation

🛠️ Technologies Used
- HTML
- CSS
- JavaScript

 📌 How to Run
1. Download or clone the repository  
2. Open `index.html` in your browser  

 ⚠️ Note
This is a frontend-only project. No backend or database is used. Authentication and data are stored locally for demonstration purposes.
